Step 12: Accessibility Issues
After running the final step in the Action 
Wizard with the Accessibility Checker, 
the results are prompted on screen for 
review. The Accessibility Checker will 
detect and prompt any remaining issues. 
These issues are identified using specific 
icons to represent each type of issue: Passed (green check mark), Needs 
Manual Check (question mark), 
and Failed (X).
•Important: any issue with a Needs Manual 
Check orFailed will need to be fixed before the 
document can be considered accessible.
